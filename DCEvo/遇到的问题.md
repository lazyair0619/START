# 跑通代码

## 1.安装torch==2.0.0+cuda117时出现报错

原因：带 +cuda117 的版本属于官方编译好的特殊二进制包，不在普通的 pip 仓库里，而在 PyTorch 官方自己的仓库里

解决：指定路径，pip 去官方的 CUDA 仓库找

    pip install torch==2.0.0+cu117 torchvision==0.15.1+cu117 torchaudio==2.0.1 --extra-index-url https://download.pytorch.org/whl/cu117

## 2.安装相关依赖时出现[Errno 28] No space left on device

原因；D盘空间充裕，但C盘空间不足以保存缓存

解决：pip cache purge清除缓存，然后用--cache-dir="D:\pip_cache"指定缓存地址

    pip install --cache-dir="D:\pip_cache" torch==2.0.0+cu117 torchvision==0.15.1+cu117 torchaudio==2.0.1 --extra-index-url https://download.pytorch.org/whl/cu117

## 3.NumPy 2.x不兼容NumPy 1.x

    pip install "numpy<2"
