 ## 判别增强模块（Discriminative Enhancer, DE）

代码中被定义为prominent类

```
  class prominent(nn.Module):
    def __init__(self, channels=0):  # 通道数（未用）
        super(prominent, self).__init__()
        self.act = nn.Sigmoid()  # Sigmoid激活
        self.act2 = nn.SiLU(inplace=True)  # SiLU激活

    def forward(self, x):
        # 前向传播：突出重要特征
        y = self.act(x)  # Sigmoid
        return self.act2(x * self.act(y.pow(2) / y.mean(dim=[2, 3], keepdim=True)))  # 加权激活
```

  $$\tilde{x}_{i} = x_{i} \times S\left(\frac{y_{i}^{2}}{\mu_{y}}\right)$$
 
  #### 平方项 ($y_i^2$) —— 极化作用：
  对特征进行平方运算。这样做的目的是拉开差距：原本数值大的（目标）会变得极大，数值小的（背景）增加不明显。
  #### 均值项 ($\mu_y$) —— 自适应阈值：
  计算当前特征图的全局平均亮度。将其作为分母，意味着这是一个相对指标。
  #### Sigmoid 激活 ($S$) —— 软掩码生成：
  通过 Sigmoid 函数将上面的比值映射到 $(0.5, 1)$ 之间的一个权重。

  ### 作用
在 C2f 模块里：它放在了特征融合的最后一步。当红外和可见光的特征通过卷积合并后，prominent 出来“把关”，决定哪些融合后的特征值得被送往下一层。

在 TransformerBlock 里：它被用来增强注意力和前馈网络的输出。
